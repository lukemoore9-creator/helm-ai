# Bridge Equipment — OOW Exam Focus

Bridge equipment covers radar, ARPA, ECDIS, AIS, compasses, echo sounder, and other SOLAS V/19 required systems. Examiners test operational knowledge, limitations, and error sources for each system.

KEY EXAM AREAS:
- Radar/ARPA — X-band vs S-band differences, gain/STC/FTC controls, CPA/TCPA alarms, trial manoeuvre, ARPA limitations
- ECDIS — safety contour vs safety depth settings, anti-grounding alarms, route checking, ZOC data quality categories
- AIS — what it broadcasts (MMSI, position, COG, SOG, heading, ROT), and that it is NOT a collision avoidance tool
- Compass — gyro errors (latitude, speed) vs magnetic (variation, deviation), cross-checking methods
- Echo sounder — measures at transducer, limitations (aeration, thermoclines)

COMMON MISTAKES:
- Not knowing the difference between X-band (3cm, better target discrimination) and S-band (10cm, better in rain)
- Confusing safety contour (ECDIS ground alarm) with safety depth (individual sounding highlight)
- Treating AIS as a collision avoidance tool — it supplements radar, never replaces it
- Forgetting ARPA limitations — steady-state assumption, target swap in congestion, requires manual checks
- Not mentioning BNWAS when asked about bridge equipment
