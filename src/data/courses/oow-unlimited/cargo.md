# Cargo — OOW Exam Focus

Cargo covers safe handling, stowage, and securing of various cargo types under SOLAS VI/VII, IMSBC, and IMDG codes. Examiners test your understanding of hazards, precautions, and regulatory requirements.

KEY EXAM AREAS:
- Bulk cargo liquefaction — Group A cargoes, transportable moisture limit (TML), flow moisture point, can test
- Dangerous goods — 9 IMDG classes, segregation levels, documentation (manifest, declaration, packing certificate)
- Container operations — VGM requirement (mandatory), CSC plate validity, reefer monitoring procedures
- Cargo securing — CSS Code principles, lashing calculations, securing for heavy weather
- Ventilation — dew point rule for deciding whether to ventilate or seal cargo holds

COMMON MISTAKES:
- Not knowing what TML means or how the can test works for detecting liquefaction risk
- Confusing IMDG segregation levels (away from, separated from, separated by compartment, separated longitudinally)
- Forgetting that VGM (verified gross mass) is mandatory before container loading
- Not mentioning the dew point rule when asked about cargo ventilation
- Mixing up Group A (liquefaction), Group B (chemical hazard), and Group C (no special hazard) bulk categories
